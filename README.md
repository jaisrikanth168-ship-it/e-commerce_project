
# QuickBuy – Grocery Shopping Web App

QuickBuy is a responsive grocery shopping web app where users can browse fruits and vegetables, view offers, and add products to the cart. It is built using HTML, CSS, and JavaScript to demonstrate basic e-commerce functionality.


## Features

- 🛒 Add products to cart
- 🔍 Search products easily
- 📱 Responsive design for mobile and desktop
- 🍎 Separate sections for fruits and vegetables
- 🏷️ Special offers section
- ⚡ Fast and simple user interface


## Installation

Follow these steps to run the project locally:

1. Clone the repository
   git clone https://github.com/your-username/quickbuy.git

2. Navigate to the project folder
   cd quickbuy

3. Open the project
   Open the `index.html` file in your browser.
